# Stack-Based-Singly-Linked-List
Singly Linked List: 
https://github.com/thabang-m-modiba/Singly-Linked-List
### When implementing a stack using singly linked list, we:
* Insert and remove elements at the head
* Do push and pop in O(1) time

## Some important methods:
* <code>push(element) // insert an element in the list</code>
* <code>Element pop() // Removes and returns the last element added on the list</code>
* <code>Element top() // Returns the last element added on the list without removing it</code>
